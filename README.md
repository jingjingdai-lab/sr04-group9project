# SR04 - Groupe 9 : Détection intelligente du trafic (YOLOv8 + Flask)

### Présentation du projet
Ce projet a été développé dans le cadre du module **SR04** à l’**Université de Technologie de Compiègne (UTC)**,  
par le **Groupe 9**.  
Il s’agit d’un système de **détection de trafic intelligent** basé sur la vision par ordinateur et l’apprentissage profond.

Le système utilise **YOLOv8** pour détecter en temps réel les véhicules à partir d’une caméra ou d’une vidéo,  
et communique avec un **serveur Flask** pour contrôler dynamiquement un feu tricolore virtuel (rouge / jaune / vert)  
en fonction du nombre de véhicules détectés.
